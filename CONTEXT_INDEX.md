# CONTEXT_INDEX

- `docs/AGENTS.md` — global instructions (style, safety, testing, context hygiene).
- `docs/architecture.md` — current architecture snapshot (navigation/data/storage/UI shells).
- `docs/sprint_status.md` — active sprint tasks, blockers, handoff/next step.
- `docs/sprint_archive/` — prior sprint status files (add per sprint close).
- `docs/decisions/` — ADR-style decisions (add per decision; keep short).

When starting a chat: read this index, then open only the files relevant to the current task (typically `docs/sprint_status.md` + any linked files).
